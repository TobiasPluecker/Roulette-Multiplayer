Detailed Description

Hier findet die komplette Lobby verwaltung statt.

Grundsätzlich werden neue Lobbys im Array lobbys gespeichert. Dadurch wird ermöglicht, dass jedes Spiel in einer Lobby unabhänig von den Anderen ist. Jede Aktion wird dann auf der jeweiligen Lobby aufgerufen.
